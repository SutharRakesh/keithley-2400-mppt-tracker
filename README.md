# Keithley 2400 MPPT Tracker

Python/NiceGUI application for J-V sweep and adaptive MPPT tracking using a Keithley 2400 SourceMeter.

## Features

- Keithley 2400 control using PyVISA
- J-V sweep measurement
- Adaptive perturb-and-observe MPPT tracking
- Live NiceGUI interface
- J-V, P-V, and MPPT power plots
- CSV and Excel export
- VISA/GPIB resource scanning
